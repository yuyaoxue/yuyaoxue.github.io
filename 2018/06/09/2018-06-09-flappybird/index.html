FlappyBird 制作进度

Posted on Jun 9, 2018

FlappyBird 制作进度：

1.创建 GitHub 仓库；

2.安装 Unity2018.1.0b12,创建 Unity 工程，导入项目资源；

3.处理图集；

4.整理整体思路，创建相关文件夹以及目前会涉及到的脚本；

5.根据背景图大小设置 Camera 的相关参数；

5.游戏模式：简单，复杂，困难

FlappyBird 制作进度 6月11号
